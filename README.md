# 心伴 MindMate — 香港大學生專屬 AI 心理健康原型

**原型網站： https://mindmatehk.netlify.app/**

心伴（MindMate）是一個為香港大學生而設的網頁原型，提供貼地易用的心理健康支援體驗。透過 AI 聊天伙伴、心情日記、本地資源及匿名社群，協助同學以親切方式建立自我照顧習慣。

## 🌟 核心特色
- **本地化體驗** — 介面採用繁體中文及廣東話語氣，整合香港熱線、大學支援及文化活動資訊。
- **AI 聊天機械人** — 全天候陪伴，提供共感而貼地的回應。
- **心情記錄 + 小日記** — 紀錄每日情緒、撰寫「備註／小日記」，繪製趨勢並隨時重溫。
- **匿名社群分享** — 一鍵將日記內容分享至社群，與同路人互相支持。
- **資源庫** — 快速瀏覽求助熱線、校園支援、本地正念練習及生活減壓點子。
- **響應式設計** — 使用 HTML/CSS 打造，兼容桌面及流動裝置。

## 📚 參考資源
- 香港心理健康熱線（撒瑪利亞會、生命熱線、社會福利署、自殺危機熱線等）
- 各大專院校輔導／健康服務（港大、中大、科大、城大、理大）
- 正念練習、呼吸放鬆、睡眠衛生等本地化建議
- 香港舒壓好去處（西九文化區、香港公園、維港海濱等）

此原型由攻讀數據科學及人工智能碩士的團隊為 JUMPSTARTER ZPIRE 參賽而製，期望為香港年輕世代提供更貼心的心理健康支援。
